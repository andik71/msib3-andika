Halo. Perkenalkan nama saya Andika Dwiyanto, saya adalah seorang mahasiswa yang sekarang ini sedang dalam semester 5 yang menempuh pendidikan S1 Teknik Informatika di Universitas Tadulako. Saya lahir tanggal 03 Mei 2001, di Kota Palu, Sulawesi Tengah. Saya merupakan anak kedua dari 6 bersaudara. Ayah saya merupakan seorang kepala teknisi di sebuah perusahaan tempat saya tinggal dan Ibu saya hanya berprofesi sebagai penjahit. Ekonomi keluarga saya tidaklah kurang dan tidak juga lebih. Kehendak tuhan saya bisa menerima beasiswa dari pemerintah.

Di luar status saya sebagai Mahasiswa, saya terkadang suka mengisi waktu luang dengan menonton Youtube atau Watch di Facebook untuk mengisi waktu, serta dengan harapan mendapatkan pengetahuan baru dan karya yang bisa saya jadikan referensi. Bicara tentang karya, saya memiliki kebiasaan untuk membuat desain dan fotografi. Disamping itu, saat libur saya sering bersama dengan teman saya untuk bermain musik di sebuah rental studio musik. Tak disangka saya juga seorang personil band sebagai Guitarist. Kesukaan saya tentang musik sudah tidak diragukan lagi, sebab selain memainkan instrument dari lagu-lagu terkenal, saya juga bisa membuat lagu itu sendiri. Hobi lain saya yaitu Memanah. Saya suka memanah sebab itu membuat saya merasa tenang saat menarik busur panah dan fokus agar anak panah dapat terkena target dengan akurat.
